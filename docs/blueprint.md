App Name: Acess11 Insights

 Core Features:

- Intelligent Audit: AI-powered accessibility analysis using axe-core to identify WCAG violations on web pages. Employs the axe-core tool to analyze the webpage and suggest potential fixes.
- Issue Display: Real-time display of accessibility issues directly in the browser popup, categorized by impact level.
- Detailed Reporting: Detailed reports including issue descriptions, WCAG success criteria, and affected HTML elements with CSS selectors.
- Auto-Scan Control: Toggle to enable/disable auto-scanning on page load.
- Privacy-Focused Design: Client-side analysis ensures user privacy; no data leaves the browser.
